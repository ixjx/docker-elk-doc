安装Kibana插件

安装Kibana插件

镜像中的Kibana主目录的名称存储在KIBANA_HOME环境变量中（在基础镜像中设置为/opt/kibana）。Kibana的插件管理脚本（kibana-plugin）位于bin子目录中，插件安装在installedPlugins子目录中。

Kibana使用用户kibana运行。为了避免权限的问题，因此建议使用kibana安装Kibana插件，使用gosu命令（参见下面的示例，并参考进一步的细节）。

下面的Dockerfile可以用来扩展基本镜像并安装最新版本的Sense plugin，这是一个与Elasticsearch的REST API交互的便利控制台：

FROM sebp/elk

WORKDIR ${KIBANA_HOME}
RUN gosu kibana bin/kibana-plugin install elastic/sense

请参阅上面构建镜像部分，以说明如何构建新的镜像。然后，可以使用与使用方法中的命令行相同的命令行基于此镜像运行容器。在http://<your-host>:5601/apss/sense（例如 http://localhost:5601/app/sense， 用于本地实例的Docker），可以访问Sense接口。
